# API resources

SmartCV stores structured, verified data that helps you generate customized CVs. Each resource represents a specific type of user-provided information.

- **[Backgrounds (`bkgds`)](../api-resources/bkgds.md)**  
  Shows career backgrounds, domains, and functional categories.

- **[Tools (`tools`)](../api-resources/tools.md)**  
  Lists skills, software, frameworks, and tool categories.

- **[Credentials (`creds`)](../api-resources/creds.md)**  
  Lists degrees, certifications, licenses, and other qualifications.

- **[Jobs (`jobs`)](../api-resources/jobs.md)**  
  Displays work experience, responsibilities, and job metadata.

- **[Portfolio (`portfolio`)](../api-resources/portfolio.md)**  
  Shows work samples linked to past jobs.

- **[Achievements (`achievements`)](../api-resources/achievements.md)**  
  Lists awards, recognition, and accomplishments linked to jobs.

Each page describes its fields, shows sample JSON, and explains how to filter the data.
